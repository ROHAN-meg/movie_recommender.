# Movie Recommendation System

## Project Objective

The Movie Recommendation System suggests similar movies based on user preferences and movie similarity.

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Features

* Recommends movies based on similarity
* Uses content-based filtering
* Fast and simple recommendation engine

## How to Run

1. Install required libraries:
   pip install pandas scikit-learn

2. Run the program:
   python movie_recommender.py

## Sample Output

Input:
Interstellar

Output:
Recommended Movies:

1. The Martian
2. Gravity
3. Arrival
4. Contact
5. Ad Astra
